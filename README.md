# 테트리스 (학습용)

HTML, CSS, JavaScript만 사용하는 브라우저 테트리스 게임 프로젝트입니다.  
빌드 도구나 외부 라이브러리 없이 바로 실행할 수 있습니다.

## 실행 방법

### 1. 파일 확인

프로젝트 폴더에 아래 파일이 있는지 확인합니다.

```
tetris-cursor/
├── index.html
├── style.css
├── script.js
└── README.md
```

### 2. 브라우저에서 열기

**방법 A — 파일 직접 열기**

1. `index.html` 파일을 더블클릭하거나
2. 브라우저 주소창에 파일 경로를 입력합니다.

   예: `file:///C:/DEV/tetris-cursor/index.html`

**방법 B — 로컬 서버 사용 (권장)**

터미널에서 프로젝트 폴더로 이동한 뒤 아래 중 하나를 실행합니다.

```bash
# Python 3
python -m http.server 8080

# Node.js (npx)
npx serve .
```

브라우저에서 `http://localhost:8080` 으로 접속합니다.

### 3. 화면 확인

다음 요소가 보이면 정상입니다.

- 10×20 격자의 게임 보드
- 점수 표시 (0)
- **시작**, **재시작** 버튼
- 키보드 조작법 안내

**시작** 버튼을 누르면 보드 상단에 테트로미노 블록(I, O, T, S, Z, J, L)이 표시됩니다.

> 현재 구현됨: 보드 렌더링, 7종 블록 정의·표시, 게임 상태 관리  
> 아직 미구현: 자동 낙하, 키보드 조작, 라인 클리어, 게임 오버

## 프로젝트 구조

| 파일 | 역할 |
|------|------|
| `index.html` | 게임 화면 구조 (보드, 점수, 버튼, 조작법) |
| `style.css` | 레이아웃, 블록 색상, CSS 변수(`--board-cols`, `--board-rows`) |
| `script.js` | 게임 로직 (보드·블록 렌더링, 충돌 검사, 블록 고정) |

## 주요 함수 (script.js)

| 함수 | 역할 |
|------|------|
| `createPiece()` | 테트로미노 블록 생성 |
| `drawPiece()` | 활성 블록을 보드에 합성 (표시용) |
| `renderBoard()` | 보드 데이터를 화면에 반영 |
| `isValidPosition()` | 블록 배치 가능 여부 검사 |
| `lockPiece()` | 활성 블록을 보드에 고정 |

## 조작법 (구현 예정)

| 키 | 동작 |
|----|------|
| ← → | 좌우 이동 |
| ↓ | 빠르게 내리기 (soft drop) |
| ↑ | 블록 회전 |
| Space | 즉시 내리기 (hard drop) |
| P | 일시정지 |
